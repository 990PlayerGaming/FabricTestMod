# Avaritia Fabric – Minecraft 1.21.11

## Summary
The purpose of this project is to remake the well-known **Avaritia** mod for current versions of Minecraft by implementing it in **Fabric**.

It has both educational and collaborative aspects:  
As I am actively learning how to create a mod using Fabric in the current version of Minecraft, it is an opportunity to get experience on how to do it.

The source code will be publicly available under the Open Source policy.

---

## 🔧 Project Goals
- Reimplement the items, mechanics, and concepts from the Avaritia mod  
- Learn how to use the Fabric API and mod Minecraft  
- Make a high-quality and extensible code base  
- Accept contributions via pull requests

---

## How to Contribute?
All pull requests are encouraged.  
No matter if you want to fix bugs or just add something new, every pull request will be highly appreciated!

If you have some thoughts that can contribute to the project, you can open an **Issue**.
